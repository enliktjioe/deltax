## How to Use

- Add raw data folder into this directory `DataCleaner/`
    - Raw data folder usually named like this: `yyyymmdd-<timestamp>`
    - It contains:
        - `images` folder
        - .h264 file
        - .json file
        - .mp4 file
- Run `python clean_json_asRCSnail.py` or `python clean_json_asRCSnail_masked.py`
